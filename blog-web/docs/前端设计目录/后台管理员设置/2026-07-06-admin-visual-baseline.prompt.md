生成一张「个人博客后台 UI 统一视觉基准图」，用于后续图生图保持三张原型图风格一致。不要做具体业务完整页面，只做统一设计语言参考。

来源模板：
- huashu-design 网页风格库：「斜切流体渐变带 Angled Fluid Gradient」
- 信息结构参考：「功能主义网格社区 Functional Brutalism」

画布：1536x1024，横向桌面 UI 设计系统风格板，像高质量产品设计稿，中文文字由图像模型原生生成。

统一视觉语言：
- 背景：近白、浅灰、轻微冷调。
- 顶部视觉锚点：一条斜切流体渐变带，颜色只用清透蓝、薄荷青、极浅珊瑚粉，年轻、轻盈、干净。
- 结构：1px 灰线、清楚网格、明确分区、小圆角、少阴影。
- 组件：后台表单、按钮、状态提示、侧边栏、顶部栏、小型弹窗。
- 图标：统一 lucide 风格细线图标感，只用于按钮、菜单和状态提示等符号化场景，不要 emoji；头像、上传预览和真实内容预览必须使用图片或图片占位，不用图标顶替。
- 强调色：蓝青渐变用于主按钮和 active 状态；红色只用于危险操作；橙色只用于 Token 未校验。
- 字体感觉：现代中文无衬线，标题有力度，正文清晰可扫读。

基准图必须包含：
1. 顶部栏样式片段：「个人博客后台」「当前管理员：梦梦（admin）」「退出登录」。
2. 侧边栏样式片段：只出现两个菜单「后台首页」「管理员设置」，管理员设置高亮。
3. 表单样式片段：用户名 admin、管理员名称 梦梦、头像地址 /files/avatar.png。
4. 按钮样式片段：保存资料、校验 Token、刷新 Token、确认退出、取消、登录后台。
5. 状态样式片段：资料已保存、无法连接后端服务、Token 未校验、有效、已刷新。
6. 弹窗样式片段：确认退出。
