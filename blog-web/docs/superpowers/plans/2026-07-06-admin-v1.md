## 后端契约

| 能力 | 方法 | 前端请求路径 | 响应 data |
|---|---|---|---|
| 管理员登录 | POST | `/admin/login` | `AdminVO`，含 `token` |
| 退出登录 | POST | `/admin/logout` | `void` |
| 获取当前管理员资料 | GET | `/admin/profile` | `AdminVO`，不含 `token` |
| 修改管理员资料 | PUT | `/admin/profile` | `AdminVO`，不含 `token` |
| 修改管理员密码 | PATCH | `/admin/profile/password` | `void` |
| 校验 Token | GET | `/admin/validate` | `{ valid: true }` |
| 刷新 Token | POST | `/admin/refresh` | `{ token: string }` |

说明：`request.ts` 的 `baseURL` 已是 `/api`，所以 `api` 文件中继续使用 `/admin/...`，不要写 `/api/admin/...`，更不能写死 `http://localhost:9090`。
